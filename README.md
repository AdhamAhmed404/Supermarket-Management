# 🛒 Supermarket Management System

A C++ supermarket management system that simulates real-world shopping and inventory operations. The system supports **Admin** and **Customer** roles, enabling product management, shopping cart functionality, order processing, and checkout. It utilizes advanced data structures including **Linked Lists, Stacks, Queues, Tries, and Binary Search Trees (BSTs)** for efficient data management and operations.

## ✨ Why This Project?

This project demonstrates how advanced data structures can be applied to solve real-world business problems. It provides an interactive shopping experience for customers while giving administrators complete control over inventory, orders, and sales tracking.

## 🚀 Features

### 👨‍💼 Admin

* Add, update, and remove products
* Manage inventory and stock levels
* Process customer orders
* View low-stock alerts
* Generate sales and revenue reports
* Track the most sold products

### 🛍️ Customer

* Browse products by category
* Search products by name
* Use autocomplete product search
* Add, remove, and update cart items
* Undo cart actions
* Checkout and place orders
* View shopping cart and order history

### ⚙️ Advanced Data Structures

* **Linked Lists** → Inventory and Cart Management
* **Stack** → Undo Functionality
* **Queue** → Order Processing System
* **Trie** → Product Autocomplete Search
* **BST** → Category Management

## 🛠️ Technologies
